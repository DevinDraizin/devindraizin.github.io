IREC Project

The Intercollegiate Rocketry Engineering Competition is a global event for universities around the world to build and launch rockets in Truth or Consequences New Mexico. My team and I built a 10ft, carbon fiber rocket that launched a micro gravity experiment 13700ft in the air. The experiment simulated micro asteroids impacting surfaces in microgravity to explore how the dust from the impact interacts to eventually form larger bodies. We designed and casted our own solid rocket motor containing potassium nitrate and sorbitol (sugar) which was used for the launch in NM. Another interesting feature was the body tube which we layed up using carbon fiber fabric, epoxy resin, and a specially designed mandrel for form.

Relational Database

This was a java framework designed to use data access models to create flexible MySQL tables for fast and extremely scalable data retrieval. This can be used to subclass larger data tables without forcing a single convention on each subclass. Because each subclass maintains a one to one relationship with the primary data table, we can use the data in each subclass to perform search and filter operations on the primary data table. Additionally, we can use hash table structures to link user specified information with a data type during runtime for ultimate flexibility when creating a subclass table.

Feel free to contact me for more information about the framework.
